Displays the statistics from ejabberd XMPP server through REST API.

== Description ==
Displays the statistics from ejabberd XMPP server through REST API (by using module mod_rest). The plugin is useful when the XMPP server is located on another machine. Easy to configure and use - just need to type REST API url and insert shortcuts on the page. Plugin can save the server statistics to the database and show them in a graph just like Munin.

= Live demo =
[Here](http://jix.im/en/statistics) you can see the statistics, generated by this plugin, from my own XMPP server.

= Translations =
* English - default

== Installation ==
This section describes how to install the plugin and get it working.

1. Install XMPP Statistics either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
